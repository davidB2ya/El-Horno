# El-Horno 🍕
***
Website where you can not only buy your favorite pizza but you can create it and name it

## Technologies 🛠️
* **React.JS**
* **Node.JS**
* **Express.JS**
* **Mongo DB**

***

## Installation 📝
***
```
git clone https://github.com/davidB2ya/El-Horno.git
cd El-Horno
npm install 
cd client/
npm run start

* open new console
cd server/
npm install 
npm run dev
```


### Pre-requirements 📋

_Only one browser is required_

```
Google Chrome
Mozilla Firefox

```
