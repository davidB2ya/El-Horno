# El-Horno 🍕
***
Website where you can not only buy your favorite pizza but you can create it and name it

## Technologies 🛠️
* **React.JS**
* **Redux**
* **Node.JS**
* **Express.JS**
* **Mongo DB**

***

## Installation 📝
***
```
git clone https://github.com/davidB2ya/El-Horno.git
cd El-Horno/
npm install 
cd server/
npm run dev

cd ..
cd client/
npm run start

open navigation http://localhost:3000/

```
